# Yeller

## Getting Started

Yeller is a real-time chat application built with Rails 5.1.5, Ruby 2.5, and Postgresql.

### Prerequisites

Install system dependencies

```
$ rbenv install 2.5.0
```

```
$ brew install postgresql
```

Install Rails dependencies

```
$ bundle install
```

Setup the development environment

```
$ bin/rails db:setup
```

Run the specs:

```
$ bin/rspec spec
```

### Installing

Give it a go!

```
$ bin/rails s
```
